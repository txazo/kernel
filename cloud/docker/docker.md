## Docker

* Docker Daemon: Docker守护进程
* Docker Client: Docker客户端
* Docker Registry: Docker仓库
* Docker Image: Docker镜像
* Docker Container: Docker容器

#### Docker Image

镜像，是一个轻量级的、独立的、可执行的包，它包含运行一个软件所需的所有东西，包括代码、运行环境、库、环境变量和配置文件

#### Docker Container

容器，是镜像的运行时实例
